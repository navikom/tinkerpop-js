# Gremlin core js


## Commands
### install
`npm install`

Dependency module install
### build

`npm run build`

### test

`npm run test`

### develop
`npm run dev` : Run develop server

You can see result in

* OS X : http://0.0.0.0:8080,

* Windows : http://localhost:8080

## List of available tasks

### clean

 `rm -f dist/*`

 Delete existing dist files
